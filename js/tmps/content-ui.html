Css 3D aplicado en aplicaciones

Css3 en 3D actualmente en 2018 en Chrome, Safari y Opera en las ultimas versiones son compatibles aplicandole los atributos para conseguir este efecto tridimensional, el atributo principal es 'transform' al cual se le puede aplicar los atributos rotate, rotateY, rotateX, rotateZ para que dicho efecto funcione tenemos que saber donde aplicar el atributo perspective siempre al padre envolvente de el/los elementos/capas a aplicar el efecto 3d, lo aplicaremos en los sgtes ejemplos:

  • Rotación Y X Z : en este ejemplo moveremos/manipularemos via "sliders" un cubo en los puntos cardinales tridimensionales, asi veremos como la orientacion, dirección en el que se mueve cada punto.
  • Capa Rotación de ejes : en este ejemplo escogeremos una capa/elemento a manipular seguidamente escogeremos un eje X Y Z, para luego manipular una capa individualmente.
  • Rotación de Perspectiva : esta definición se puede traducir como eje de perspectiva, punto de referencia el cual nuestro elemento se movera segun los cardinales como eje escogido; ejemplo 50% 50% (X Y).
